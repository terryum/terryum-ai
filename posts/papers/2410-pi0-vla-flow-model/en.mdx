## TL;DR

A robot foundation model combining a pre-trained VLM with flow matching-based action generation, trained on 10,000+ hours across 7 robots and 68 tasks to perform complex dexterous manipulation from laundry folding to box assembly with a single model.

## Problem

Traditional robot learning has focused on training task-specific policies individually, facing three core challenges: data scarcity, limited generalization, and fragile robustness. Recent VLA models like RT-2, Octo, and OpenVLA attempted multi-task learning, but discrete token-based action generation proved insufficient for high-frequency dexterous manipulation (50Hz). This paper applies the LLM pre-training/fine-tuning paradigm to robotics at scale, pre-training on large-scale data from diverse robot platforms and then efficiently transferring to complex downstream tasks.

## Key Idea

**VLM + Flow Matching Action Expert**: A pre-trained 3B VLM (PaliGemma) serves as the vision-language backbone, while action generation is handled by a flow matching-based continuous action expert rather than discrete tokens. This enables high-frequency continuous action chunks up to 50Hz and applies the LLM-style pre-training → post-training two-phase recipe to robotics.

Key components:

- **Flow Matching Action Generation**: Learns a vector field transforming noise to action sequences ($H=50$ steps), with 10-step Euler integration at inference
- **Action Expert**: A separate 300M parameter expert network for robot proprioception/action tokens, enabling efficient inference
- **Blockwise Causal Attention**: Three-block attention over image·language / robot state / noisy actions, with bidirectional attention within action tokens
- **Large-scale Pre-training**: 7 robot configurations, 68 tasks, 10,000+ hours of demonstration data (OXE + proprietary π dataset)

<Collapsible title="Detailed Method">

π₀'s architecture and training pipeline consists of three main parts.

**1. Flow Matching Action Generation**: Given observation $\mathbf{o}_t = [\mathbf{I}^1_t, ..., \mathbf{I}^n_t, \ell_t, \mathbf{q}_t]$, the model generates action chunks $\mathbf{A}_t = [\mathbf{a}_t, ..., \mathbf{a}_{t+H-1}]$. During training, it learns a denoising vector field from noisy actions $\mathbf{A}^\tau_t = \tau \mathbf{A}_t + (1-\tau)\epsilon$, with the loss function:

$$L^\tau(\theta) = \mathbb{E}_{p(\mathbf{A}_t|\mathbf{o}_t), q(\mathbf{A}^\tau_t|\mathbf{A}_t)} \| \mathbf{v}_\theta(\mathbf{A}^\tau_t, \mathbf{o}_t) - \mathbf{u}(\mathbf{A}^\tau_t | \mathbf{A}_t) \|^2$$

At inference, 10-step forward Euler integration: $\mathbf{A}^{\tau+\delta}_t = \mathbf{A}^\tau_t + \delta \mathbf{v}_\theta(\mathbf{A}^\tau_t, \mathbf{o}_t)$

**2. Action Expert Architecture**: VLM language/vision processing is separated from robot action generation, with VLM tokens and action tokens using independent weights. The action expert uses a reduced configuration (width=1024, mlp_dim=4096) for fast inference. Flow matching timestep $\tau$ is injected via sinusoidal positional encoding.

**3. Pre-training → Post-training Two-Phase Recipe**: Pre-training uses diverse but variable-quality large-scale data to learn broad physical capabilities and error recovery behaviors. Post-training uses small amounts of high-quality task-specific data to learn refined execution. Pre-training alone doesn't produce fluent strategies, while post-training alone doesn't teach error recovery — both phases are essential.

**Timestep Sampling**: $\tau$ is sampled from a shifted beta distribution $p(\tau) = \text{Beta}(\frac{s-\tau}{s}; 1.5, 1)$ ($s=0.999$), emphasizing lower (noisier) timesteps.

**Cross-robot Training**: Different action dimensions (6-DoF to 17-DoF) are zero-padded to 18 dimensions, with task-robot combination weights adjusted by $n^{0.43}$ to balance data imbalance.

## Key Results

- **Out-of-box (zero-shot) evaluation**: π₀ significantly outperformed OpenVLA and Octo across 5 tasks, achieving ~95% on shirt folding vs OpenVLA ~10%
- **Fine-tuning efficiency**: Pre-trained π₀ achieved up to **2x** performance over training from scratch on most tasks, with pre-training advantage amplified on harder tasks
- **Complex long-horizon tasks**: On 7 tasks requiring 5-20 minutes (laundry folding, box assembly, egg packing, etc.), pre-trained + post-trained model achieved **3-4x** performance over training from scratch

## Achievements and Limitations

### What's New

- First implementation of 50Hz high-frequency dexterous manipulation in a VLA framework via flow matching continuous action generation
- Largest-scale robot foundation model pre-trained on 10,000+ hours across 7 robot configurations and 68 tasks
- Achieved long-horizon dexterous tasks (5-20 minutes, including laundry folding and box assembly) via end-to-end learning — described by the authors as "the longest dexterous tasks in the end-to-end robot learning literature"

### Limitations

- **Dataset composition understanding**: Optimal data combinations and data requirements for near-perfect performance remain unclear
- **Transfer scope uncertainty**: Transferability to significantly different domains like autonomous driving, navigation, and legged locomotion remains unverified
- 🤖 **Proprietary data**: 90.9% of pre-training data is proprietary (π dataset), severely limiting reproducibility for external researchers
